Transaction Data Enrichment & Analysis
Overview

This project demonstrates how transactional data can be combined, transformed, and analyzed using SQL.

What this project shows
Joining multiple tables
Creating derived fields using CASE statements
Aggregating data for insights
Detecting unusual transactions
Files

queries/02_spending_summary.sql → aggregation example
queries/03_anomaly_detection.sql → anomaly detection example
data/sample_data.sql → sample dataset
